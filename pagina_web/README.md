# Página Web Interactiva: Nuestra Historia

Esta página web es una experiencia interactiva romántica creada con HTML, CSS y JavaScript puro.

## Estructura de Archivos

- `index.html`: Página principal con la estructura HTML.
- `styles.css`: Estilos CSS para diseño moderno y responsive.
- `script.js`: Lógica JavaScript para la interactividad.
- `img/`: Carpeta para imágenes (botones y galerías).
- `video/`: Carpeta para videos de las galerías.

## Instrucciones de Uso

1. **Imágenes de Botones**: Coloca las imágenes para los 3 botones iniciales en `img/` con nombres:
   - `boton1.jpg`
   - `boton2.jpg`
   - `boton3.jpg`

2. **Contenido de Galerías**: Agrega tus imágenes y videos en las carpetas correspondientes:
   - Imágenes: `img/gal1_1.jpg`, `img/gal1_2.jpg`, etc.
   - Videos: `video/gal2_1.mp4`, `video/gal2_2.mp4`, etc.

3. **Actualizar Arrays en script.js**: Edita los arrays `galerias` en `script.js` para incluir todas tus rutas de archivos.

4. **Abrir la Página**: Abre `index.html` en un navegador web.

## Funcionalidades

- Menú inicial con 3 botones (2 incorrectos, 1 correcto).
- Segundo menú con 5 galerías independientes.
- Modales con navegación para imágenes y videos.
- Diseño responsive para móvil y escritorio.
- Animaciones suaves y efectos hover.

## Notas

- Asegúrate de que los archivos multimedia estén en los formatos compatibles (JPG, PNG para imágenes; MP4, WebM, OGG para videos).
- Los videos se reproducen con controles HTML5.
- El mensaje ofensivo se muestra en un modal elegante.
